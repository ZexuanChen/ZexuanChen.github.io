pip和conda常用命令

整理一下pip和conda的常用命令。

pip

pip是python的包管理器，用来安装第三方库。

pip官方文档的描述：

pip is the package installer for Python. You can use it to install packages from the Python Package Index and other indexes.

如果你是在虚拟环境中使用的python，如conda管理的环境，或者在python官网下载的python，都是自带pip的。如果没有内置pip，可以用以下方法下载：

  • ensurepip方法。python有一个内置的模块叫ensurepip，可以用来安装pip。

  • **get-pip.py方法。**执行python自带的安装pip的文件get-pip.py。

检查是否下载成功：

在cmd中输入py -m pip --version，出现pip x.y.z from ... (python 3.n.n)则下载成功。

pip常用命令如下：

  • 查看pip和python所在目录路径：where pip 和 where python。

    如果有多个python，按照环境变量的优先级来进行排序，在安装了多个python时可以通过路径知道哪个python是默认的。如果要更改默认的python环境，需要更改设置里面的环境变量的配置，有环境变量>无环境变量，系统环境变量>用户环境变量。

  • 安装包：pip install package_name。

    pip install 一般会有默认的安装目录，可以通过 python -m site 进行查看。当然也可以对默认目录进行修改，通过 pip install --target=path_name package_name。

  • 查看已安装的包：pip show package_name。

  • 显示默认路径的全部包：

  • 升级包：pip install --upgrade package_name。

  • 卸载包：pip uninstall package_name。

conda

conda是一个开源的软件包管理系统和环境管理系统，它可以在不同的操作系统上安装和管理多个软件包和环境。conda通常不用手动下载，内置在anaconda中，作为虚拟环境的包管理工具。

anaconda：

Anaconda是一个开源的Python和R语言的发行版本，用于计算科学，Anaconda致力于简化软件包管理系统和部署。Anaconda透过Conda进行软件包管理，并拥有许多适用于Windows、Linux和MacOS的数据科学软件包。

anaconda主要特点是集成了很多科学计算包，但它主要还是用于创建虚拟的python开发环境。

以下是一些常用的conda命令：

  • 安装软件包：conda install package_name
  • 更新软件包：conda update package_name
  • 创建环境：conda create --name env_name package_name
  • 激活环境：conda activate env_name
  • 关闭环境：conda deactivate
  • 查看所有环境：conda env list
  • 删除环境：conda remove --name env_name --all
