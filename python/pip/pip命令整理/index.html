整理一下pip的知识点。

pip简介

pip是python的包管理器，用来安装第三方库。

pip官方文档：

pip is the package installer for Python. You can use it to install packages from the Python Package Index and other indexes.

pip下载

如果你是在虚拟环境中使用的python，如conda管理的环境，或者在python官网下载的python，都是自带pip的。

ensurepip方法

python有一个内置的模块叫ensurepip，可以用来安装pip。

get-pip.py方法

执行python自带的安装pip的文件

检查是否下载成功

在cmd中输入py --version，若出现版本号，如python 3.n.n，则安装成功。或者输入py -m pip --version，出现pip x.y.z from ... (python 3.n.n)。
